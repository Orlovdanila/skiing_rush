# Исправление ошибок типизации

1. Запусти `npx tsc --noEmit` 
2. Проанализируй ВСЕ ошибки типизации
3. Исправь каждую ошибку по очереди
4. После каждого исправления снова запусти tsc для проверки
5. Повторяй пока все ошибки не будут исправлены

ВАЖНО: Не используй `any` для обхода ошибок - исправляй типы правильно.
