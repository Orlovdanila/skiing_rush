# Проверка проекта

Выполни полную проверку проекта:

1. Запусти `npx tsc --noEmit` для проверки типов
2. Если есть ошибки - выведи их и предложи исправления
3. Запусти `npm run build` для проверки сборки
4. Сообщи результат

Если всё прошло успешно, выведи ✅ Проверка пройдена
Если есть ошибки, выведи ❌ и список проблем
