# Поиск AICODE-маркеров

Выполни поиск всех AICODE-маркеров в проекте:

```bash
grep -rn "AICODE-" src/ server/
```

Для каждого найденного маркера:

## AICODE-ASK (вопросы)
1. Выведи вопрос пользователю
2. Дождись ответа
3. После ответа:
   - Обнови код согласно ответу
   - Удали или замени маркер на AICODE-NOTE если нужен контекст

## AICODE-NOTE (заметки)  
1. Выведи список всех заметок
2. Отметь устаревшие (если есть)
3. Предложи удалить неактуальные

Если маркеров нет — сообщи "✅ Нет открытых AICODE-маркеров"
