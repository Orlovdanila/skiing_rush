# Добавить ошибку в AGENTS.md

Ошибка/решение: $ARGUMENTS

1. Прочитай `AGENTS.md` (единственный источник правды)
2. Найди секцию "❌ НЕ делай" или "✅ Делай"
3. Добавь новую запись в соответствующую секцию
4. Сохрани файл
5. Подтверди добавление

ВАЖНО: Обновляй только `AGENTS.md`, НЕ `CLAUDE.md` или `.cursor/rules/`
